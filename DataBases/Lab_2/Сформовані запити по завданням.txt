ВАРІАНТ 10. Автозаправна станція
Реалізувати такі запити до таблиць БД, створених у лабораторній роботі №1:

1. Вивести алфавітні списки прізвищ та ініціалів всіх клієнтів автозаправної станції.

SELECT Lastname AS Прізвище, CONCAT(SUBSTRING(Firstname, 1, 1), '.', SUBSTRING(Father, 1, 1), '.') AS Ініціали FROM клієнт;

2.Вивести номери дисконтів клієнтів, які купили бензин марки 'N' M-го числа.

SELECT DISTINCT НомерДисконту FROM продаж WHERE МаркаБензину = 'A-95' AND DATE(ДатаПродажу) = '2024-02-20';

SELECT DISTINCT НомерДисконту FROM продаж WHERE МаркаБензину = 'А-95' AND DATE(ДатаПродажу) = '2024-02-19';

3. Для кожної марки бензину видати максимальне і мінімальне значення його вартості.

SELECT  МаркаБенизну, MAX(ЦінаПродажу_грн) AS Максимальна_вартість, MIN(ЦінаПродажу_грн) AS Мінімальна_вартість FROM `вартість бензину`GROUP BY МаркаБенизну;

4. Для кожної дати продажів вивести сумарне значення проданого бензину марки 'N'.

SELECT ДатаПродажу, МаркаБензину, SUM(КількістьЛітрів_л) AS Сум_прод_л FROM `продаж` GROUP BY ДатаПродажу, МаркаБензину ORDER BY ДатаПродажу;

5. Вивести номери дисконтів клієнтів автозаправної станції, які купили за один раз максимальне кількість літрів бензину.

SELECT НомерДисконту FROM продаж WHERE КількістьЛітрів_л = (SELECT MAX(КількістьЛітрів_л) FROM продаж);

більш інформативний
SELECT НомерДисконту, МаркаБензину, КількістьЛітрів_л FROM продаж WHERE КількістьЛітрів_л=(SELECT MAX(КількістьЛітрів_л) FROM продаж);

ще більш інформативний за типами бензину
SELECT МаркаБензину, НомерДисконту, MAX(КількістьЛітрів_л) AS Макс_кіл_л FROM продаж GROUP BY МаркаБензину;
